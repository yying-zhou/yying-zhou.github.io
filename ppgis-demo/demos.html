3DGS-PPGIS Value Annotation Instrument

3D Gaussian Splatting / Participatory Mapping / Multispecies Justice

A browser-based tool for annotating place-based values directly onto a photorealistic 3D scene. Participants tap a location in the scene, select a value type (recreational, aesthetic, wildlife habitat, avoidance), and add a free-text note. Pins are stored with 3D scene coordinates and dwell time.

Built as the Phase 2 research instrument for a PhD proposal targeting Prof. Nora Fagerholm's Sustainable Landscape Systems Research Group at the University of Turku. The instrument enables a controlled A/B comparison between 2D map-based PPGIS and 3D immersive PPGIS — testing whether representational modality affects which values participants report and where they locate them spatially.

RESEARCH QUESTION

Does the shift from 2D map to 3D immersive scene systematically surface different value types — in particular, wildlife habitat and spatially marginal values invisible in plan view?

TECHNICAL STACK

Single-file HTML · Canvas 2D (demo) → Spark.js / THREE.js (production) · Supabase + PostGIS · No build step · Deployable as GitHub Pages URL

STATUS

Prototype — simulated scene. Replace scene.splat + scene.glb from Luma AI to connect to a real photogrammetric capture.
